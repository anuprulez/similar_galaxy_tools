Galaxy similar tools visualizer

Find similar Galaxy tools by learning optimal similarity mixtures of multiple sources of annotations.

The text similarity is computed using a neural network approach called as Doc2Vec. It learns a dense vector for each document. The similarity scores are combined using an optimizer (Gradient Descent).
Sources of annotations:
  • Input and output file types
  • Name, description and EDAM
  • help text
